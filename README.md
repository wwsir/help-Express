# 快递帮取小程序

一个基于微信小程序的快递代取服务平台，提供互助式的快递收取服务。

## 功能特色

### 主要功能

- **我帮取快递**：查看并接取附近的快递代取需求
- **我有快递**：发布快递代取请求，寻找帮取服务
- **用户登录**：支持微信授权登录和游客登录
- **个人中心**：查看订单、收益统计和个人设置

## 页面说明

### 1. 首页 (index)

- 展示4个主要功能菜单
- 根据登录状态显示不同内容
- 快速导航到各个功能模块

### 2. 登录页面 (login)

- 微信授权登录
- 游客模式登录
- 服务特色展示

### 3. 我帮取快递 (help-deliver)

- 浏览可接取的快递列表
- 按距离和价格筛选
- 接单确认和联系信息

### 4. 我有快递 (have-package)

- 发布快递代取需求
- 填写详细的快递信息
- 管理已发布的需求

### 5. 个人中心 (profile)

- 用户信息展示
- 统计数据查看
- 功能设置入口

## 技术栈

- **框架**：微信小程序原生开发
- **语言**：JavaScript, WXML, WXSS
- **数据管理**：本地存储 + 全局状态管理
- **UI组件**：微信小程序原生组件

## 开发说明

### 环境要求

- 微信开发者工具
- 微信小程序开发者账号

### 快速开始

1. 使用微信开发者工具打开项目
2. 配置小程序 AppID（在 project.config.json 中）
3. 编译并预览小程序

### 功能扩展

当前版本为基础功能实现，可扩展的功能包括：

- 真实的后端API集成
- 地图定位服务
- 支付功能
- 推送通知
- 订单状态追踪
- 用户评价系统

## 注意事项

1. **数据模拟**：当前版本使用模拟数据，实际部署需要对接后端服务
2. **图标资源**：tabBar图标需要准备相应的图片资源
3. **权限申请**：某些功能可能需要申请相应的微信小程序权限
